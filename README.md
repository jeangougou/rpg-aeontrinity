# rpg-aeontrinity
utils for the campaign

## setup

Requires any version of node. tested on (v16.13.1)
To install node on windows use Chocolatey and nvm.

## Usage 

```
> npm install
// generates the character sheet in pdf
> npm run charSheet
```
